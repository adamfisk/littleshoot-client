1
Download the LittleShoot Plugin

 If you haven't already downloaded it, you can download the LittleShoot P2P Plugin from here.
2
Select Where to Download the Installer

 First, save the installer to your computer. It's often a good idea to save it to your desktop, where you can find it easily afterwards.

3
Double Click the Downloaded File

The file will be called "LittleShoot-[version].dmg". When you open it, you should see the image below. When you see this image, follow the instruction to double click the "LittleShoot.mpkg" icon. This should open automatically if you're using Safari.

4
Follow the Installer Instructions

The installer will run you through the steps of installing LittleShoot. Continue along each step. You should see the images below, although they may appear slightly differently if you've previously installed an earlier version of LittleShoot on your computer.



5
OK, We're Done!

If everything went according to plan, you should see the screen below. At this point, LittleShoot is running on your computer as a helper application for your browser. You can now go back to the LittleShoot web site and start:
  1. Publishing Your Files
  2. Searching for Files
  3. Downloading Files
A new browser window should open to the LittleShoot site automatically. If it doesn't, click here.



As always, we sincerely hope you enjoy using LittleShoot. If you have any problems at all or any suggestions, please don't hesitate to visit our forums.
