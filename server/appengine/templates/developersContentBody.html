LittleShoot is open source and based on open protocols. To get started quickly, you can do any of the following:
  • Check out the code and run LittleShoot
  • Join the developer mailing list
  • Have a look at outstanding bugs/tasks or report a new one
  • See what bugs users are seeing in the field
LittleShoot includes code in many languages, including:
  • Java
  • Python
  • C++
  • JavaScript
  • ActionScript
We also use the following frameworks and libraries, with many more not listed:
  • Google App Engine
  • Google App Engine Patch
  • Amazon Web Services (Most of 'em!)
  • Django
  • Spring
  • Dojo
  • JQuery
  • Lib Torrent
  • Jetty
  • NSIS
  • PURE JavaScript Templating
LittleShoot uses protocols ranging from SIP/STUN/TURN/ICE to BitTorrent, Gnutella, and the old classic, HTTP. In all cases, LittleShoot strives to use the best tool for the job, whether it's a framework, external open source project, protocol, or programming language. If you think we can do better along any one of these vectors, let us know on the developer mailing list.

The breadth of what constitutes "LittleShoot" means there are many opportunities for you to contribute your skills. Whether you're a coding Ninja in JavaScript, Python, C++, or Java, we've likely got a battle you could join. We strive to keep code at a high quality and well organized, and we constantly encourage your feedback and improvements.

Also keep in mind we're completely self-funded and do this because we think it's how the Internet should work. Translation: we need all the help we can get!
