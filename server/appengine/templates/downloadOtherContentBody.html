Expand the TGZ

You're on Linux or another Unix, so we trust you know what you're doing. Our installer is currently just a tgz that you can expand in, say, /usr/local or wherever you like using:

sudo tar xzvf LittleShoot-[version].tgz

We'll eventually include a more sophisticated script that will likely do things like prompt you if you'd like us to create an entry in init.d, but for now you have to run it manually.

First, navigate to the LittleShoot directory with:

cd LittleShoot-{version}

Then you can start LittleShoot with:

nohup ./run.sh &

or without nohup, without backgrounding, etc as you prefer.

Once LittleShoot is running, go back to the web site here.

If you'd like to lend a hand with creating a more sophisticated installer for general Unix flavors or for any specific variety, feel free to pipe up over on the forums.
